Learning becomes far more effective when you test your understanding. This extension turns Grokipedia reading into interactive quizzes — instantly generated from the content you are already viewing.

Key Features

  • On‑device AI — works offline after the model downloads
  • Custom difficulty levels and question count
  • Reads only the current tab after user action
  • No data collection, storage, or external transmission
  • Clean quiz interface with scoring and explanations

How it works

  1. Open any Grokipedia article in Chrome.
  2. Click the floating “Quiz Me” button or the extension icon.
  3. Select difficulty and number of questions.
  4. The extension extracts visible text from the page (only after your click) and sends it to Chrome’s built‑in on‑device AI.
  5. The AI generates a quiz locally — no servers, no data sent out.
  6. Answer questions and get your score instantly.

Permissions Used

  • activeTab — Access only the current page after a user clicks “Generate Quiz.”
  • scripting — Extract visible article text to feed into the on‑device AI.
  • contextMenus — Optional right‑click shortcut for faster access.

Host permissions are restricted only to Grokipedia domains.

FAQ

Does this extension send my data online?
No. All processing happens on your device using Chrome’s built‑in AI.

Is this an official Grokipedia extension?
No. This is an independent tool and is not affiliated with, endorsed by, or associated with Grokipedia.
